🚀 DSA Arcade – Gamifying Data Structures & Algorithms

DSA Arcade is an interactive learning platform that transforms Data Structures & Algorithms (DSA) into fun, engaging, and gamified experiences.
Built with Python (Pygame + Tkinter) and a prototype website, this project helps learners strengthen their problem-solving skills through games and challenges.

✨ Features

🎮 Gamified Learning – Learn DSA concepts through interactive levels.

🧩 Progressive Difficulty – Beginner, Intermediate, and Advanced levels.

💻 Integrated Code Editor – Write, edit, and test your solutions in-app.

🌐 Prototype Website – A preview of the platform for better accessibility.

🎨 Engaging UI – Fun arcade-style interface for students.

🛠️ Tech Stack

Python

Pygame – Game development

Tkinter – UI & Code Editor integration

Web Prototype – HTML, CSS, JavaScript (for future extension)

🚩 How to Run

Clone the repository

git clone https://github.com/your-username/DSA-Arcade.git
cd DSA-Arcade


Install dependencies

pip install pygame


Run the app

python app.py
